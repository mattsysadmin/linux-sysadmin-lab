\# Linux Sysadmin Lab



A personal lab environment documenting my journey toward the Linux Foundation Certified System Administrator (LFCS) certification.



\## Environment

\- Host: Windows 11 (i5 12th Gen, 16GB RAM, RTX 3050)

\- Virtualization: VirtualBox 7.x

\- Guest OS: Ubuntu Server 22.04 LTS



\## Repository Structure

\- `notes/` - Concept notes and theory

\- `labs/` - Step by step lab documentation

\- `scripts/` - Bash scripts written during labs
