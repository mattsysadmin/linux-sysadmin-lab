\# Lab 03 - WordPress Installation



\## Date

2026-04-23



\## Objective

Install and configure WordPress on the LAMP stack.



\## Steps Completed

\- Downloaded WordPress latest from wordpress.org

\- Extracted and moved to /var/www/html/wordpress

\- Set correct ownership (www-data) and permissions (755)

\- Configured wp-config.php with database credentials

\- Created Apache virtual host configuration

\- Enabled wordpress site and rewrite module

\- Disabled default Apache site

\- Completed WordPress web installer



\## Verification

\- Accessed WordPress installer at http://127.0.0.1:8080

\- Successfully logged into WordPress dashboard
